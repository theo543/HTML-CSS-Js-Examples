Web Languages


The Internet is a global system of networks that provide a variety of functions. Cross network communication is governed by protocols e.g. File Transfer Protocol, Simple Mail Transfer Protocol, HyperText Transfer Protocol, etc.

The most common use of the internet is the World Wide Web, a collection of interconnected web pages that follow the HyperText Transfer Protocol.

Web pages are programmed using Web Languages:

  • HTML - HyperText Markup Language
  • CSS - Cascading Style Sheets
  • JavaScript

This web site uses all three languages.
