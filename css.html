CSS


CSS (Cascading Style Sheets) is a web language used to style web pages. It is added to a web page by linking a CSS file using the "link" tag. Every web page on this site contains the following tag: <link rel="stylesheet" href="style.css">. It tells the browser to load the style.css file and apply it to this web page.

A CSS Rule is some CSS code that is only applied to certain HTML tags.
A CSS Selector determines which tags a rule is applied to.

If an element is targeted by several rules, the rules will be merged. If the rules conflict, more specific rules will override less specific rules. A rule that makes all text blue will be overridden by a rule that makes a specific paragraph black. A rule that makes text bold will merge with a rule that makes a specific paragraph red.

Examples of CSS rules in style.css:

Target Description
Root elements Removes default margins.
All elements Sets the font to Arial.
Body element Sets variables that control the color of the background and the page element.
Body element when it has "dark-theme" class Changes color variables to dark colors.
Links and elements with "emphasis" class while body has "dark-theme" class. Increases brightness to be easier to read with dark theme.
Tables and all components of tables Adds a border to this table. Touching borders are set to merge.
First cell of each row of all tables Centers text.
"page" class Centers, sets the background color, expands vertically to fill the viewport, adds a shadow, adds margin and padding. On small screens, it expands horizontally.
"title" class Centers the page title.
"emphasis" class Makes text class italic and blue.
Elements with "code" class Adds a radial gradient background, rounded corners, padding, makes text green, makes the element always display as a rectangle (inline block), slightly shrinks the text to keep it from touching the gradient borders.
Elements with "NA-box" class. This class sets a gray background and centers text.
The element with ID "css-test" Hides the element for testing the CSS toggle script.
Element with class of "links" This class, in conjunction with the following rule, horizontally aligns the elements inside the container element. Also adds a bottom border.
Elements inside element with class of "links" Horizontally aligns the elements in conjunction with the previous rule.
The link element with ID of "current" Disables the link and makes it bold.
Page root Makes the vertical scrollbar always show even if not needed so there is no alignment change between short pages and long pages.
Table cells with class "code-cell" Similar to "code" class. Shrinks text more.
Elements with "highlighted" class Highlights element and adds a border with a transition. Overrides all other rules.
Elements with "hide" class Hides element. Overrides all other rules.
Element with "reset-button" id Adds rounded corners, black background, white text, sets cursor to pointer and adds margin and padding. Fixed position, unaffected by scrolling. Is hidden by a "hide-reset-button" class, with a transition.
Element with "theme-button" id Sets fixed position in bottom right corner.
Label inside element with "theme-button" id Sets display to fixed size block, replaces content with a moon image.
Label inside element with "theme-button" id, if body has "dark-theme" class Replaces content with a moon image with inverted color.
Input inside element with "theme-button" id Hides it (checkboxes cannot be directly styled, clicking the label is used instead).
