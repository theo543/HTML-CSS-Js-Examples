JavaScript


JavaScript is a scripting language used to make interactive web pages. It is inserted into a web page using the script tag, either directly into the HTML file or loaded from a file or remote server.

JavaScript interacts with the user by altering the Document Object Model. JavaScript receives input by attaching event listeners (functions) to elements e.g. listening to click events on a button.

The following script removes a button when it is clicked:

var button = document.getElementById("mybutton");
function eventlistener(){
    button.parentElement.removeChild(button);
}
button.addEventListener("click", eventlistener);


The HTML and CSS pages on this web site use JavaScript to toggle CSS properties.

This site uses 3 scripts:

  • htmlToggle.js - Inserts radio buttons into the second cell of each row of the table with the ID "toggletable", unless the cell has the "ignore" class. Applies "hide" and "highlighted" CSS classes to the elements specified in the first cell on click. Displays the "resetter" element when at least one element is modified. If <body> or <html> are hidden the script will reset after one second.
  • themeToggle.js - Inserts a checkbox that toggles the dark theme class on <body>. Styled by CSS. Stores the current theme in localStorage. The initial value is decided by the OS theme if it is not set in localStorage. Updates when the OS theme is changed.
  • csstoggleselective.js - adds event listeners to all cells with the "toggle" class that add/remove CSS classes to the page root element. Each toggleable CSS rule only selects elements if the page root does not have an unique "disabler" class. This class is specified as an attribute in each toggle cell.
  • csstoggleall.js - adds an event listener to a specific button to toggle all CSS on the page on/off. Works by removing/inserting the HTML tag that inserts CSS into the page. If csstogglesselective.js is also loaded all disabler classes are removed when the tag is reinserted.
